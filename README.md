# Bright-TV-Analysis
This project analyses the Bright TV viewership data to uncover patterns in viewers behavior, peak viewing times and potential business insights.


## Aim
To provide the Bright TV's CEO with an analysis that will assist in growing the company's subscription base and assist the CVM (Customer Value Management) team in meeting this year’s objective. 

## Objectives
-Understand viewers behaviour
-Identify popular content
-Identify peak viewing hours
-Segment viewers based on time and behaviour
-Provide actionable insights for content scheduling

####Tools used
-Miro for planning
-Canva for project planning
-SQL(Databricks) for data cleaning, quering and aggregating data
-Excel for creating pivot charts and generate insights
-Canva for presentation
